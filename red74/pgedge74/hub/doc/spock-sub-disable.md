## SYNOPSIS
    ./nodectl spock sub-disable SUBSCRIPTION_NAME DB <flags>
 
## DESCRIPTION
    Put a subscription on hold and disconnect from provider.
 
## POSITIONAL ARGUMENTS
    SUBSCRIPTION_NAME
    DB
 
## FLAGS
    -i, --immediate=IMMEDIATE
        Default: False
    -p, --pg=PG
        Type: Optional[]
        Default: None
